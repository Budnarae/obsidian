---

**디스트럭처링 할당 문법**은 배열의 값 또는 객체의 속성을 풀어서 별개의 변수로 쓸 수 있게 해주는 자바스크립트 표현식이다.

## 객체 디스트럭처링

```javascript

const person = {
	first: "Alberto",
	last: "Montalesi",
};

const {first, last} = person;

```

디스트럭처링을 이용하여 person이 가진 속성에 접근함과 동시에 해당 속성 이름으로 변수 선언이 가능함을 알 수 있다.
