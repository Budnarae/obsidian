다음과 같이 인자 3개의 기본값이 설정되어 있는 함수가 있다고 가정하자

```javascript

function calculatePrice({total = 0, tax = 0.1, tip = 0.05,} = {})
{
	return total + (total * tax) + (total * tip);
}

const bill = calculatePrice({tip: 0.15, total: 150});

```
