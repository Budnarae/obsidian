#tag

*Protocol Family*

---

[[socket | 소켓]]이 사용할 프로토콜의 부류정보를 가리켜 '프로토콜 체계'라 하며, 프로토콜 체계의 종류는 다음과 같다.

| 이름     | 프로토콜 체게(Protocol Family) |
| -------- | ------------------------------ |
| PF_INET  | IPv4 인터넷 프로토콜 체계      |
| PF_INET6 | IPv5                               |
