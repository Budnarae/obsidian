#docker

*Dockerfile*

---

**Dockerfile**은 [[이미지]]를 직접 만들기 위한 설정 파일이다.

Dockerfile 스크립트에는 토대가 될 이미지나 실행할 명령어 등을 기재한다.

이 파일을 호스트 컴퓨터의 이미지 재료가 들어있는 폴더(위치는 어디라도 상관없다.)에 넣는다.
재료 폴더에는 그 외 컨테이너에 넣을 파일을 함께 둔다. 실제 컨테이너를 만들 필요는 없다.

### 주요 Dockerfile 인스트럭션

| 인스트럭션 | 내용                      |
| ---------- | ------------------------- |
| FROM       | 토대가 되는 이미지를 지정 |
| ADD        |                           |
| COPY       |                           |
| RUN        |                           |
