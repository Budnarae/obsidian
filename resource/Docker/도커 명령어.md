#docker

*docker command는 상위 커맨드와 하위 커맨드로 이루어져 있다.*

---

[[도커]]는 **도커 명령어**를 사용하여 조작한다.

도커 명령어는 일반적으로 아래의 형식을 취한다.

`docker <상위 커맨드><하위 커맨드> <옵션> <대상> <인자>`

| 요소        | 의미                                                           | 예시                              |
| ----------- | -------------------------------------------------------------- | --------------------------------- |
| 상위 커맨드 | 도커를 이루는 여러 요소 중 무엇을 명령의 대상으로 삼을 지 규정 | container, image, network, volume |
| 하위 커맨드 |                                                                |                                   |
