*temporary object*

---

함수의 반환형이 [[클래스]]이고, 함수가 값을 반환할 때, 메모리 공간에 **임시로 객체를 생성하여** 반환을 받는 쪽이 참조할 수 있도록 한다. 이를 **임시 객체**라고 한다.

외부에서 객체의 멤버함수를 호출하기 위해 필요한 것은 다음 세 가지 중 하나이다.
- 객체에 붙여진 이름
- 객체의 참조 값(객체 참조에 사용되는 정보)
- 객체의 주소 값

임시객체가 생성된 위치에는 임시객체의 참조 값이 반환된다.
