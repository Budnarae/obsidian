#language #cpp 

*class template*

---

함수를 [[템플릿]]으로 정의하듯이 클래스도 템플릿으로 정의가 가능하다. 그리고 이렇게 정의된 템플릿을 가리켜 **클래스 템플릿(class template)**이라 하며, 이를 기반으로 컴파일러가 만들어 내는 클래스를 가리켜 **템플릿 클래스(template class)**라 한다.

템플릿 클래스는 아래의 예제와 같이 만들 수 있다.

```cpp

template <typename T>
class Point
{
	private :
		T xpos, ypos;
	public :
		Point(T x = 0, T y = 0)
}

```
