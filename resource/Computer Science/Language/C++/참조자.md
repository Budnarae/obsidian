변수명은 할당된 메모리 공간에 이름을 붙이는 것과 같다. 참조자는 이미 변수명이 붙어있는 메모리 공간이 제 2의 별칭을 붙이는 것과 같다. 변수명을 통해 메모리 공간의 데이터를 조작할 수 있는 것과 마찬가지로, 참조자를 통해서도 메모리 공간의 데이터를 조작할 수 있다.

참조자는 아래와 같이 선언될 수 있다.

~~~cpp

#include <iostream>

using namespace std;

int main(void)
{
	int num1=1020;
	int &num2=num1;

	num2=3047;
	cout<<"VAL: "<<num1<<endl;
	cout<<"REF: "<<num2<<endl;

	cout<<"VAL: "<<&num1<<endl;
	cout<<"REF: "<<&num2<<endl;
}

~~~
