*static*

---

기존의 C 언어에서 static 키워드는 다음의 의미를 가진다.

- 전역 변수에 선언된 static의 의미 -> 선언된 파일 내에서만 참조를 허용하겠다는 의미
- 함수 내에 선언된 static의 의미 -> 한번만 초기화되고, 지역변수와 달리 함수를 빠져나가도 소멸되지 않는다.

C++에서 static 키워드는 추가적으로 다음의 의미를 가진다.

- static 멤버변수는 'zmffk'
