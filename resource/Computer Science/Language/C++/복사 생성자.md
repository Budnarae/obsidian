C++ 언어에서는 아래와 같은 형식으로 객체 간 복사가 가능하다.

```cpp

class SoSimple
{
	private :
		int num1;
		int num2;
	public :
		SoSimple(int n1, int n2) : num1(n1), num2(n2) { }
		void ShowSimpleData()
		{
			cout<<num1<<endl;
			cout<<num2<<endl;
		}
};

int main(void)
{
	SoSimple sim1(15, 20);
	SoSimple sim2 = sim1; //sim2 객체가 sim1 객체의 구성 요소(멤버 변수)를 복사
	
	sim2.ShowSimpleData();

	return (0);
}

```

위 예제에서, 객체의 복사는 다음 구문에서 일어난다.

`SoSimple sim2 = sim1;`

이러한 형태의 구문은 묵시적 변환이 일어나 실제로는 다음의 형태로 해석된다.

`SoSimple sim2(sim1);`
