| 토폴로지 | 형태                                             | 프로토콜 |
| -------- | ------------------------------------------------ | -------- |
| 스타형   | 허브를 중심으로 모든 컴퓨터가 연결되어 있는 형태 | 이더넷   |
| 버스형         |                  한 개의 케이블에 모든 컴퓨터가 연결된 형                                |          |

1. 스타형 : . 이더넷에서 주로 사용됨
