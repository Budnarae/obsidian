# get_next_line

_파일 입출력 기초_

다음의 기능과 형식을 가진 `get_next_line`이라는 함수를 만드는 과제

```c

// 입력값 : 파일 디스크립터
// 반환값 : 파일로부터 문장 하나를 추출하여 반환.
char *get_next_line(int fd);

```

n번째로 get_next_line을 호출하면 파일의 n번째 문장을 반환한다.
또한, 여러 개의 파일 디스크립터를 동시에 처리할 수 있다.
