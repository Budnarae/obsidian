## qr code js 코드 상에 적용하는 법

```javascript

QRCode.toCanvas(qrcodeUrl, { width:256 } , function (error, canvas) {
        if (error) {
          console.error(error);
        }
        else {
          target.appendChild(canvas); // qr code가 담겨있는 캔버스를 원하는 대상에 추가시킨다.
        }
      });

```
