## 피사체 휘도 (Subject Luminance)

**휘도(Luminance)**는 **특정 방향으로 방출되는 빛의 밝기**를 나타내는 물리량입니다.

### 핵심 개념

- **단위**: cd/m² (칸델라 per 제곱미터) = nit
- **의미**: 관찰자가 보는 표면의 밝기
- **방향성**: 특정 방향에서 본 밝기 (viewing angle 중요)

### 예시

```
태양 표면: ~1,600,000,000 cd/m²
컴퓨터 모니터: 100~300 cd/m²
달 표면: ~2,500 cd/m²
흰 종이 (실내): ~100 cd/m²
```

### 사진/렌더링에서

```
피사체 휘도가 높을수록:
- 더 밝게 보임
- 더 짧은 노출 시간 필요
- 더 작은 조리개 사용 가능 (큰 F-number)
```

### 수학적 정의

```
L = dΦ / (dA × dΩ × cos θ)

L  = 휘도
Φ  = 광속 (luminous flux)
A  = 면적
Ω  = 입체각 (solid angle)
θ  = 표면 법선과의 각도
```

## 조도 (Illuminance)

**조도**는 **표면에 입사하는 빛의 양**을 나타내는 물리량입니다.

### 핵심 개념

- **단위**: lux (lx) = lumen/m²
- **의미**: 표면이 받는 빛의 세기
- **방향성 없음**: 모든 방향에서 오는 빛의 총량

### 예시

```
직사광선 (한여름): ~100,000 lux
흐린 날 야외: ~10,000 lux
사무실 조명: ~300~500 lux
보름달 빛: ~0.1 lux
별빛: ~0.001 lux
```

### 사진/렌더링에서

```
조도가 높을수록:
- 표면이 더 밝게 보일 가능성 (반사율에 따라 다름)
- 더 빠른 셔터 스피드 가능
- 더 낮은 ISO 사용 가능
```

### 수학적 정의

```
E = dΦ / dA

E  = 조도
Φ  = 광속 (luminous flux)
A  = 면적
```

## 휘도 vs 조도: 핵심 차이

|항목|휘도 (Luminance)|조도 (Illuminance)|
|---|---|---|
|**의미**|표면이 방출/반사하는 빛|표면이 받는 빛|
|**관점**|관찰자가 보는 밝기|광원에서 오는 빛의 양|
|**방향성**|있음 (특정 방향)|없음 (전체 입사광)|
|**단위**|cd/m² (nit)|lux (lx)|
|**측정 대상**|발광체, 반사면|조명받는 표면|

### 비유로 이해하기

```
조도 (Illuminance):
"이 책상에 얼마나 많은 빛이 비추고 있는가?"
→ 입사하는 빛의 양

휘도 (Luminance):
"이 책상이 내 눈에 얼마나 밝게 보이는가?"
→ 반사되어 나오는 빛의 밝기
```

## 둘의 관계

### 반사율을 통한 연결

```
L = E × ρ / π

L = 휘도 (cd/m²)
E = 조도 (lux)
ρ = 반사율 (albedo, 0~1)
π = 램버트 반사 가정
```

**예시:**

```
조도 500 lux의 사무실에서
흰 종이 (반사율 90%):
L = 500 × 0.9 / π ≈ 143 cd/m²

검은 종이 (반사율 5%):
L = 500 × 0.05 / π ≈ 8 cd/m²
```

## 사진/카메라에서의 의미

### 노출 결정

```
적정 노출 = f(피사체 휘도, ISO, 셔터 스피드, 조리개)

노출 값(EV) = log₂(L × S / K)

L = 피사체 휘도
S = ISO 감도
K = 반사 상수 (보통 12.5)
```

### 실전 예시

**맑은 날 야외 촬영:**

```
조도: ~100,000 lux (태양빛이 지면에 내리쬠)
↓
흰 셔츠 휘도: ~28,000 cd/m² (반사율 90% 가정)
↓
적정 노출: f/16, 1/125초, ISO 100
```

## 렌더링에서의 활용

### PBR (Physically Based Rendering)

```glsl
// 조도 (입사광)
vec3 irradiance = lightColor * lightIntensity;

// 표면 반사 (BRDF 계산)
vec3 reflectance = BRDF(normal, viewDir, lightDir);

// 휘도 (출사광, 최종 픽셀 밝기)
vec3 luminance = irradiance * reflectance;
```

### HDR 렌더링

```
휘도 범위:
- LDR: 0~255 (8비트)
- HDR: 0~무한대 (floating point)

tone mapping으로 HDR 휘도를 LDR로 변환
```

## 요약

|개념|한 문장 정의|
|---|---|
|**조도**|표면이 **받는** 빛의 양|
|**휘도**|관찰자가 **보는** 표면의 밝기|

**관계:**

```
빛 → 조도 (표면 입사) → 반사 → 휘도 (눈/카메라 도달)
```

**사진에서:**

- 조도: 조명 설정 (스튜디오 조명의 lux 값)
- 휘도: 노출 결정 (측광계가 측정하는 값)

**렌더링에서:**

- 조도: Direct lighting (광원에서 표면으로)
- 휘도: 최종 픽셀 색상 (표면에서 카메라로)
